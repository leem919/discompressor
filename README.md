# Discompressor
Video compressor for Discord

# Usage
1. Import video file
2. Select size to compress below
3. Resulting video can be uploaded to Discord

# Building
1. Install Python 3 from [python.org](https://python.org)
2. Install the dependencies with `pip install -r requirements.txt`
3. Run `pyinstaller --noconsole --clean --onefile --icon=blcompress.ico --add-data "blcompress.ico;." discompressor.py`
